# 🔍 Basic Network Sniffer

This project is a simple **Network Sniffer** built using Python and Scapy. It captures live network packets and displays useful information such as source IP, destination IP, protocol type, and port numbers.

---

## 🚀 Features

- Captures real-time network packets
- Displays:
  - Source IP address
  - Destination IP address
  - Protocol (TCP/UDP)
  - Source and destination ports
  - Timestamp of packet capture
- Filters only TCP traffic for cleaner output
- Stops automatically after capturing 20 packets

---

## 🛠️ Technologies Used

- Python
- Scapy Library
